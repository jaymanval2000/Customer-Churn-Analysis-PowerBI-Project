
## Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analyzing telecom customer data to identify patterns and factors influencing customer churn.  
The interactive dashboard provides insights into customer behavior, service usage, payment methods, and contract types.

The dashboard was developed using Microsoft Power BI to transform raw data into meaningful business insights.

## Dashboard Features

- Customer churn distribution
- Churn rate analysis by tenure
- Internet service usage (Fiber Optic, DSL, No Internet)
- Payment method analysis
- Monthly and total charges analysis
- Customer contract type analysis
- Customer service ticket trends
- Interactive filters and slicers for better exploration

## Tools & Technologies Used

- Microsoft Power BI
- Data Modeling
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

## Key Insights

- Customers with **Month-to-Month contracts** show higher churn rates.
- **Fiber Optic users** have a higher churn rate compared to other services.
- Customers with **short tenure** are more likely to churn.
- Payment methods such as **Electronic Check** show higher churn patterns.
